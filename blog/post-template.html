这是一篇用于测试排版和展示样式的模板文章。在编写博客时，可以直接复制本文件的 HTML 结构，并修改相应的内容，快速产出符合网站视觉风格的漂亮文章。

一、排版结构与标题层级

在正文写作中，建议一级标题仅留给文章主标题，正文内部的子标题从 <h2> 开始，它将自动被 TOC (目录) 模块捕获并高亮显示。

这是一段普通段落文本。我们在 blog.css 中专门为正文设定了 1.0625rem (17px) 的阅读字号，以及 1.75 的行高，以确保在长时间阅读技术长文时眼睛不易疲劳。

1.1 三级标题演示

如果文章内容极其详实，可以使用三级标题 <h3> 细分段落，它同样支持 TOC 目录抓取与跳转。

二、引用块与列表样式

“这是一个精美的毛玻璃引用块（Blockquote）示例，带有左侧渐变指示条。适合放置名人名言、关键结论、或者开发中需要提醒读者的重要提示。”

在编写清单或流程时，可以使用无序列表和有序列表：

  • 列表项一：支持多级列表嵌套
  • 列表项二：使用方块状的项目符号，并涂上了全局主题的激活色
  • 列表项三：保持合适的行间距，避免内容过于拥挤

2.1 任务清单 (Checklist) 演示

任务清单是记录学习计划、项目进度的极佳方式：

  • 已完成的任务一（会有划线和变淡效果）
  • 正在进行的任务二（空心框，高亮框线）
  • 计划中的任务三

三、代码块展示与复制

对于技术博客，代码段的显示极为关键。我们为 <pre><code> 容器封装了一层 macOS 风格的灰色磨砂窗口。左上角带有标志性的红黄绿三色控制点，右上角则集成了一键复制按钮：

example.py 
import torch
import torch.nn as nn

# 这是一个最小的神经网络结构示例
class SimpleMLP(nn.Module):
    def __init__(self, input_dim=10, hidden_dim=32, output_dim=2):
        super(SimpleMLP, self).__init__()
        self.net = nn.Sequential(
            nn.Linear(input_dim, hidden_dim),
            nn.ReLU(),
            nn.Linear(hidden_dim, output_dim)
        )
        
    def forward(self, x):
        return self.net(x)

# 实例化并输出模型
model = SimpleMLP()
print(model)

点击右上角的 “Copy Code”，文本将被自动复制到您的剪贴板，按钮也会随之进行过渡动画，带来细腻的交互体验。
